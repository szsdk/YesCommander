# YesCommander

This is a python-based bash command snippet tool.

## Install

```
pip install .
```

## Usage

```
yc
```

### Shortcut

- `enter`: execute a command
- `ctrl-n`/`down`/`tab`: select the next command
- `ctrl-p`/`up`/`shift-tab`: select the previous command
- `ctrl-d`: copy the command
